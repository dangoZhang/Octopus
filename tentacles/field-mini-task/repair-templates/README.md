Field mini task repair templates live here so Octopus can evolve task-specific Feed suppliers without changing the stable Rust core. The current seed set has 24 templates: three per parallel field pack. `run_field_mini_task` only loads these templates, records the executed template path in Feed metadata, and falls back to a generic partial result when no template exists.

Each `*.pyfrag` file owns one narrow `field_pack` plus `field_mini_task` implementation and starts with its own `if field == ...` guard. `tools/check_repair_templates.py` compiles and executes every template in a temporary session and requires a satisfied `field_result`. Keep fragments auditable and small enough for harness evolution to rewrite safely.
