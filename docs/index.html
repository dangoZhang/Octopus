Independent-thinking tools with a clean agent brain. The model asks for cognition; an evolvable harness supplies the feed.

Core

Rust kernel

The stable base owns Need, Feed, Feedback, memory state, and route learning.

Thinking tentacles

Each tentacle owns brain prompt, permissioned tool metadata, runtime code, and explicit evolution surfaces.

Code-as-harness

SWE, computer-use with browser diagnostics, repo-maintainer, and write-and-run are agent tool-combo seeds. octopus-json-v1 lets prompt, meta, code, and policy evolve across runtimes.

Quick Install & Use

cargo install --git https://github.com/dangoZhang/Octopus --locked --package octopus-core --bin octopus
octopus --version
octopus doctor
octopus demo dangoZhang/Octopus
octopus init
octopus skills
octopus providers
octopus provider status
octopus install swe-agent
octopus install json-feed
octopus chat "build a clean-brain agent"
octopus think swe-agent observe README.md
octopus need observe .
octopus pet
octopus doctor
octopus beat 200
octopus pet
tmp=$(mktemp -d)
(cd "$tmp" && octopus scaffold my-feed python)
(cd "$tmp" && octopus probe my-feed observe README.md)
