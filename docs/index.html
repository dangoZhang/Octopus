Keep the main agent focused on the goal. Let local tentacles handle files, browsers, commands, checks, and messy tool work.

Quick Install & Use

curl -fsSL https://dangozhang.github.io/Octopus/install.sh | sh
octopus --version
octopus start --open

start --open prepares state, seed tentacles, heartbeats, and opens the native app. Use octopus start on headless machines.

octopus doctor
octopus first-run "make this repo easier to use"
octopus first-run --live "make this repo easier to use"
octopus routes observe .
octopus beat 200
octopus pet
