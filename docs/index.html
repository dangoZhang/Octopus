A Smaller Brain That Moves Faster

Stable kernel

The Rust base keeps the loop small, stores memory state, and learns which route works.

Local nervous systems

Inspired by cephalopod arms, each tentacle owns its prompt, permissions, tool metadata, runtime code, and editable surface.

Agent tool suites

SWE, computer-use diagnostics, repo maintenance, harness repair, and write-and-run ship as editable seeds that can be inspected and improved.

Quick Install & Use

cargo install --git https://github.com/dangoZhang/Octopus octopus-core --locked --bin octopus --force
octopus update
octopus start --open

start --open prepares state, seed tentacles, heartbeats, and opens the native app. Use octopus start on headless machines.

octopus doctor
octopus first-run "make this repo easier to use"
octopus first-run --live "make this repo easier to use"
