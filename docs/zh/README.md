4. 继续读 [架构说明](architecture.md)，理解 clean brain、Need、Feed、tentacle、heartbeat。
5. 最后看 [product gap](../product-gap.md) 和 [real-machine gate](../real-machine-test.md)，判断当前 release gate 和下一步缺口。
