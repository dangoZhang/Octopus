工具会思考，大脑才干净。

Octopus 是一个 Rust agent kernel。模型只说自己需要什么；环境学会怎样满足它；触手在执行工具前先思考。

大脑轻了，工具强了

轻内核

Rust 基座保持小，只管目标、记忆、结果和路线学习。

会思考的工具

每条触手带自己的提示词、权限、工具描述、运行代码和可修改边界。

可改的工具组合

SWE、带浏览器诊断的 computer-use、repo-maintainer、harness-repair、write-and-run 都是初始 agent 工具套件，可读、可测、可进化。

状态一眼可见

活着

内核和 chat loop 正常运行时，像素章鱼会变色。

记忆在动

召回、压缩、遗忘都可以变成颜色状态。

工具在学

路由学习和工具执行可以给出可见反馈，同时保持主脑干净。

cargo install --git https://github.com/dangoZhang/Octopus octopus-core --locked --bin octopus
octopus --version
octopus doctor
octopus demo dangoZhang/Octopus
octopus --lang zh bootstrap
octopus --lang zh skills
octopus --lang zh providers
octopus provider status
octopus install swe-agent
octopus install harness-repair-agent
octopus goal set "build a clean-brain agent"
octopus chat "build a clean-brain agent"
octopus brain "what should the brain ask next?"
octopus brain --goal --live --save "make the goal sharper"
octopus brain --live "what should the brain ask next?"
octopus brain --live --save "what should the brain ask next?"
octopus explore "what should the brain ask next?"
octopus explore --save "what should the brain ask next?"
octopus needs
octopus think swe-agent observe README.md
octopus need observe .
octopus traces
octopus report
octopus preflight
octopus preflight script
octopus preflight record
octopus pet

tmp=$(mktemp -d)
(cd "$tmp" && octopus scaffold my-feed python)
