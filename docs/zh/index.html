工具会思考，大脑才干净。

Octopus 是一个 Rust agent kernel。模型只表达认知需求；harness 用数据决定供给；触手在执行工具前先思考。

核心洞察

干净大脑

主模型只说需要查证、复现、比较、记忆、遗忘或执行，不背工具细节。

触手大脑

触手由 brain prompt、工具 meta、runtime code 和可进化面组成。

Code-as-harness

SWE、带浏览器诊断的 computer-use、repo-maintainer、write-and-run 是初始 agent 工具组合；json-feed 验证 octopus-json-v1 能跨 runtime 演化。

章鱼变色

Heartbeat

像素章鱼变色表达内核和 chat loop 是否存活。

Memory beat

召回、压缩、遗忘时，状态颜色可以变化。

Harness beat

路由学习和工具执行可以变成可见反馈，不进入主脑。

打开像素桌宠

快速安装与使用

cargo install --git https://github.com/dangoZhang/Octopus --locked --package octopus-core --bin octopus
octopus --version
octopus doctor
octopus demo dangoZhang/Octopus
octopus --lang zh init
octopus --lang zh skills
octopus --lang zh providers
octopus install swe-agent
octopus chat "build a clean-brain agent"
octopus need observe .
octopus pet
octopus --lang zh doctor
octopus --lang zh beat 200
tmp=$(mktemp -d)
(cd "$tmp" && octopus scaffold my-feed python)
(cd "$tmp" && octopus probe my-feed observe README.md)

完整流程。打开 HTML App。内核保持小，SDK 和触手围绕它进化。
