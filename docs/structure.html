核心小，触手可改。

Rust 保持 Goal、Need、Feed、状态和观察链路稳定。触手、field packs、worker harness 留在可进化代码区。

三层边界

稳定核心

  • 干净大脑：Goal/Mem/Need/Feed
  • Need -> Feed 传输和轨迹
  • LLM provider、状态、权限、release gate

可进化触手

  • tentacles/ 存工具 prompt/meta/code
  • field-packs/ 存领域任务
  • 默认 Go worker，可按环境演化

观察产品

  • 桌宠只读状态，不遥控 agent
  • App 只让用户改 Goal
  • Docs 解释安装、使用、结构

主路径

crates/octopus-core/src/brain_loop.rs

干净大脑 loop。

crates/octopus-core/src/manifest_runtime.rs

触手侧智能执行入口。

crates/octopus-core/src/evolution_driver.rs

一次 harness 自进化驱动。

crates/octopus-core/src/state_report.rs

只读状态、field pool、环境缺口报告。

tentacles/field-mini-task/

领域任务的可编辑 harness。

field-packs/

math、search、code、SWE、research、computer-use、IB、robotics 等领域包。

desktop/pet/OctopusDesktopPet.swift

macOS 桌宠观察器。

docs/app.html

本地 Goal/Need/Feed App。
