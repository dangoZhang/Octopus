Tools should think. The brain should stay clean.

Octopus is a small Rust kernel for agents that ask for cognition, then receive data-driven feed from an evolvable harness.

Insight

Old loops bind thought to action. Octopus separates cognitive need from implementation. The model says what it needs. Tentacles decide how to satisfy it.

Shape

Tentacle brain

A tentacle is brain prompt, tool metadata, code implementation, and evolution policy, so execution can think without loading the main brain.

Color change

A visual pet/status layer can express heartbeat, memory, harness, blocked, or success state outside the kernel.

Three hearts

Heartbeat checks liveness. Memory beat evolves context. Harness beat evolves routes.

Mechanism

Need -> Feed -> Feedback is the mechanism. Chat feels like refining a goal; the harness stores those refinements and feeds them back as evidence.

MIT licensed. Rust kernel, Python SDK, evolvable harness.
